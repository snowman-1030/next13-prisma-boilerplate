# Markdown Syntax Cheat Sheet

This Markdown file demonstrates various features of Markdown syntax:

## Headings

You can create headings using the '#' symbol. The number of '#' symbols
represents the heading level.

## Code Snippets

### Inline Code

To display inline code, use single backticks: `code`.

### Code Blocks

To show a block of code, use triple backticks:

```js filename="demo.js" {3} copy
let a = 1;

console.log(a);
```
